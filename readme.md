# SK8KREW

Social media platform for Skater's.


## Get Started

Clone the project

```bash
  git clone https://github.com/gokulselvan-r/sk8krew-ui
```

Go to the project directory

```bash
  cd sk8krew-ui
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start 
  npm run android
```
